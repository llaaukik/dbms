MongoDB - 1
MongoDB Queries: Design and Develop MongoDB Queries using CRUD operations. (Use CRUD operations, SAVE method, logical operators etc)
1️⃣ Switch/Create Database
use LibraryDB

​
2️⃣ Create Collection and Insert Data (CRUD - Create)
db.Library.insertMany([
  { Book_ID: 1, Book_Name: "C Programming", Author: "Pratik", Copies: 5 },
  { Book_ID: 2, Book_Name: "MongoDB Basics", Author: "Rhutik", Copies: 3 },
  { Book_ID: 3, Book_Name: "Node.js Guide", Author: "Om", Copies: 4 },
  { Book_ID: 4, Book_Name: "Advanced C", Author: "Ganesh", Copies: 2 },
  { Book_ID: 5, Book_Name: "MongoDB Advanced", Author: "Pratik", Copies: 6 }
]);

​
3️⃣ Read / Find Data (CRUD - Read)
3.1 Find all documents
db.Library.find();

​
3.2 Find with logical operator (Copies >= 4)
db.Library.find({ Copies: { $gte: 4 } });

​
3.3 Find with OR (Author Pratik OR Copies >= 5)
db.Library.find({ $or: [ { Author: "Pratik" }, { Copies: { $gte: 5 } } ] });

​
4️⃣ Update Data (CRUD - Update)
4.1 Update a single field
db.Library.updateOne(
    { Book_ID: 2 },
    { $set: { Copies: 10 } }
);

​
4.2 Save method behavior (insert if not exist, update if exists)
db.Library.save({ Book_ID: 6, Book_Name: "Java Basics", Author: "Om", Copies: 7 });
db.Library.save({ Book_ID: 2, Book_Name: "MongoDB Basics", Author: "Rhutik", Copies: 15 });

​
First save → inserts new document
Second save → updates existing document with Book_ID: 2
5️⃣ Delete Data (CRUD - Delete)
5.1 Delete one document
db.Library.deleteOne({ Book_ID: 4 });

​
5.2 Delete multiple documents
db.Library.deleteMany({ Copies: { $lt: 5 } });
